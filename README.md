# Processor Environment Variables

Add the following configuration to your environment file or export them before running the application:

```env
CORE_DATABASE_URL=sqlite+aiosqlite:///./core.db
PC_DATABASE_URL=sqlite+aiosqlite:///./pc.db
REDIS_HOST=localhost
REDIS_PORT=6379
REDIS_PASSWORD="(|_0U|)S0u|_\/2"
MQTT_HOST=localhost
MQTT_PORT=1883
```
